# 🏔️ Hunza Valley - Tourism Landing Page

A sleek, modern, and highly aesthetic landing page dedicated to exploring the breathtaking beauty of Hunza Valley, Gilgit-Baltistan. Built using **HTML** and **Tailwind CSS**.

## ✨ Features
- ⚡ **Modern Typography:** Clean headings and editorial-style layout.
- 🖼️ **Aesthetic Grid Elements:** Beautiful image cards showcasing key spots like Attabad Lake and Khunjerab Pass.
- 📱 **Fully Responsive:** Optimized design for all device viewports.

## 🛠️ Tech Stack
- **HTML5**
- **Tailwind CSS**
